# Math Library - William Cronin
Performs a desired math operation using two integers (num1 and num2) 
# Operations available:
* '+' - Add (num1 + num2)
* '-' - Subtract (num1 - num2)
* '*' - Multiply (num1 * num2)
* '/' - Divide (num1 / num2)
* '%' - Modulus (num1 % num2)
* '<' - Leftshift (num1 << num2)
* '>' - Rightshift (num1 >> num2)
* '&' - And (num1 & num2)
* '<' - Or (num1 | num2)
* '>' - Xor (num1 ^ num2)
* '&' - Inverse (~num1)
